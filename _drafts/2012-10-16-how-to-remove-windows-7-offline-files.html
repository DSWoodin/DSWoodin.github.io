--- layout: post title: How to remove Windows 7 Offline files date: '2012-10-16T04:48:00.000-07:00' author: syswow64 tags: - Windows 7 modified_time: '2012-12-31T06:42:36.343-08:00' blogger_id: tag:blogger.com,1999:blog-4452544758446808584.post-1358173384004284068 blogger_orig_url: https://www.syswow64.co.uk/2012/10/how-to-remove-windows-7-offline-files.html --- http://social.technet.microsoft.com/Forums/en/w7itproui/thread/9c89236e-a315-4755-b642-070c120a0448 (I tested this on Win7 and it worked for me):

1. Navigate to the following location in the registry:
HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services\Csc\Parameters

2. Create a new DWORDValue called FormatDatabase , with the value 1
3. Reboot (the new key we created will delete itself after rebooting along with the Offline cache)
