--- layout: post title: SCCM Client Certificate (PKI) Value is None date: '2016-03-16T02:12:00.003-07:00' author: syswow64 tags: modified_time: '2020-01-17T04:20:27.673-08:00' blogger_id: tag:blogger.com,1999:blog-4452544758446808584.post-3211203497762269551 blogger_orig_url: https://www.syswow64.co.uk/2016/03/sccm-client-certificate-pki-value-is_47.html --- SCCM Client Certificate (PKI) Value is None

Stopping WMI service
Stopping CCMExec

SC Delete any sccm services (ccmexec, smstsmgr, cmecservice, ccmsetup)
Delete C:\windows\ccm, C:\windows\ccmsetup, C:\windows\ccmcache, C:\Windows\SMSCFG.ini

Go into regedit and remove:

HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\CCM
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\CCMSetup
HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\SMS

Then restart WMI, and reinstall the client. You shouldn't need a reboot to complete this.

Once this has been done the client will install and pick up the cert.
