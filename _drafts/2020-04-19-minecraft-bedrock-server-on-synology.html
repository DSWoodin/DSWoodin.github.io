Install Docker on Synology NAS

Open the Docker app.

Go to Registry and search for marctv/minecraftbedrockserver (see https://hub.docker.com/r/marctv/minecraftbedrockserver for more details on this). Highlight the entry in the search results and click "Download".

Go to Image and highlight the newly downloaded server image and click Launch to create container. Give the container a name, like "bedrock-server".Tick "Enable resource limitations" and tick CPU Priority "high", and set the memory limit to about 1 GB of memory (at least -- but leave some for the system as well...) Click "Advanced settings" and tick "Enable auto-restart". Then "Volume" and "Add Folder". (This is to point to a folder outside of the container where the world is persisted between potential rebuilds of the container.) Chose an already existing folder somewhere on your server, or create a new one. You can also optionally create a couple of configuration files in the same manner (but click "Add file") -- "server.properties", for, well, server properties (an example is here) and "permissions.json" for user permissions, for example (from the doc accompanying the bedrock server):
 
[
    {
        "permission": "operator",
        "xuid": "451298348"
    },
    {
        "permission": "member",
        "xuid": "52819329"
    },
    {
        "permission": "visitor",
        "xuid": "234114123"
    }
]

(To get the users xuid you can monitor the terminal of the running docker container as users log in).
