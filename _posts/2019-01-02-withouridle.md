# *Variable ( পরিবর্তনশীল )* 
প্রোগ্রামিং এ ভেরিয়েবল বলতে জায়গা বুঝানো হয় যেখানে ডাটা বা নাম্বার রাখা যায় !
```sh 
name = 'X'
print(name)
```
# *Condition ( শর্ত )*
উদাহরণ ঃ 
```sh
age = 17
if age < 18:
  print('Not eligible')
else:
  print('Eligible')
```
# *Function ( কাজ )*
উদাহরণ ঃ 
```sh
def Func(num1 + num2):
  return num1 + num2 
 
Func(10,12)
```
